
# EditPixel - Simples Page Application 

SPA para crear sprite de forma agil. Escrita en HTML5, VanillaJS, cssVanilla.
Se trata de un prueba de concepto para crear un SPA solo usando recursos basico.

---

## 🚀 Cómo usar

Ejecuta el siguiente comando en la terminal (asegúrate de tener el compilador de C# instalado):

```sh
run.bat 54321
```
Esto llama la aplicacion **Express** para levantar un SimplesHttpServer y llama al google Chrome para que paresca una PWA ..

---

## 🔧 Dependencias

.NET Framework 4.0 o superior(Para el Express.exe).
Google Chrome instalado en la ruta por defecto (C:/Program Files/Google/Chrome/Application/chrome_proxy.exe).
